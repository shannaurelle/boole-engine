Truth Table

The truth table of an expression have the following rules in this context

  1. The first column contains all the possible value of the inputs expressed in a bitstring where the first letter has its value on the first number on the bitstring, second letter on the second number on the bitstring, and so on.
  2. The number on the result column is the value if the whole expression is evaluated based on the input bitstring at the same row.
  3. 1 means true and 0 means false
p q result
0 0 0
0 1 1
1 0 0
1 1 1

For example, in the given truth table there are four readings based on the four rows above. Note that row 1 starts just after the header which has the column names p, q, and result.

  • Row 1: if p is false and q is false, then the result is false.
  • Row 2: if p is false and q is true, then the result is true.
  • Row 3: if p is true and q is false, then the result is true.
  • Row 4: if p is true and q is true, then the result is true.
